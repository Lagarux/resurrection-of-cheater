# 📸 Resurrection of Cheater (RoC)
**Resurrection of Cheater**, ekran üzerinden anlık metin yakalama (OCR), yapay zeka tabanlı analiz ve çok dilli çeviri yeteneklerine sahip gelişmiş bir masaüstü asistanıdır. Özellikle akademik materyallerin hızlı analizi için tasarlanmıştır.

## 🚀 Öne Çıkan Özellikler
- **Sniper Tool:** Ekranın istenilen bölgesini seçip anında metne dönüştürün.
- **AI Analysis:** Google Gemini-3-Flash entegrasyonu ile metinleri saniyeler içinde analiz edin ve özetleyin.
- **Stealth Mode (WinAPI):** Windows işletim sistemi seviyesinde "odak çalmayan" pencere mimarisi sayesinde diğer uygulamaların (örn. tarayıcılar) odağını bozmadan çalışır.
- **OCR Hassasiyeti:** Tesseract-OCR altyapısı ile Türkçe ve İngilizce karakterlerde yüksek doğruluk.
- **Zengin Markdown Önizleme:** AI yanıtlarını görsel olarak şık ve okunabilir bir formatta sunar.
- **TTS & Translation:** Metinleri seslendirin veya anında diller arası çeviri yapın.

## 🛠️ Kullanılan Teknolojiler
- **Dil:** Python 3.10+
- **Arayüz:** CustomTkinter (Modern Dark UI)
- **OCR:** Tesseract-OCR
- **AI:** Google Gemini API
- **Sistem:** Windows API (ctypes)
- **Ses & Çeviri:** gTTS, Deep Translator
